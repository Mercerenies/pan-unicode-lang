Pancode

Welcome to the documentation for Pancode, the pan-Unicode programming language written by Mercerenies! Pancode is a stack-based esoteric programming language written with the simple goal in mind: to properly utilize all of the glorious Unicode characters available to us in a way that other languages have failed to do.

Note that this language (and, indeed, this documentation) is still a work-in-progress. There's a very basic tutorial available below to whet your appetite, but check back here soon and there shall be more comprehensive documentation listing all of the available functions. If you're feeling particularly ambitious, you can poke around the source code and figure it out for yourself, or you can simply check back later.
