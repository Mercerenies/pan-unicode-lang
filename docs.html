Pancode

Welcome to the documentation for Pancode, the pan-Unicode programming language written by Mercerenies! Pancode is a stack-based esoteric programming language written with the simple goal in mind: to properly utilize all of the glorious Unicode characters available to us in a way that other languages have failed to do.

Note that the Pancode source is hosted on GitHub.
